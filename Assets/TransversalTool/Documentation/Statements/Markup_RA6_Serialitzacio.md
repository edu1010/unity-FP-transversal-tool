# Llenguatges de marques RA6 - Serialització JSON de partida

Completa un fitxer JSON de partida perquè pugui ser serialitzat/deserialitzat de forma fiable.

Cal definir correctament:

- Dades del jugador.
- Nivell actual.
- Monedes recollides.
- Vides i estat de finalització.

El fitxer ha de ser JSON vàlid i consistent, perquè en una activitat posterior es guardarà a base de dades i es recuperarà.
