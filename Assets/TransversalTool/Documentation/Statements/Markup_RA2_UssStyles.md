# Llenguatges de marques RA2 - Estils USS (semblant a CSS)

Unity UI Toolkit utilitza fitxers **USS**, un llenguatge d'estils molt semblant a CSS.

Partint de la plantilla, completa els estils d'un HUD:

- Color i mida del títol.
- Estil d'una etiqueta de puntuació.
- Estil visual d'un botó principal.

Valida que la sintaxi sigui correcta i que els selectors siguin coherents amb els noms de classe indicats.
