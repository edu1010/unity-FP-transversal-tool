# Programació RA1 - Variables bàsiques en moviment

Partint d'un script de moviment ja funcional, modifica i prova variables de tipus `int`, `float` i `bool`.

Objectius mínims:

- Ajustar la velocitat (`float`) i observar canvis de moviment.
- Utilitzar una variable de vides (`int`) i decrementar-la en una prova simple.
- Activar o desactivar una funcionalitat amb un interruptor (`bool`), per exemple invertir controls o habilitar sprint.

Lliura el codi comentat i una breu explicació de què has observat en cada tipus de variable.
