# Transversal Tool (Unity Editor)

Obre la finestra des de `Tools/Transversal Tool/Package Generator`.

1. Selecciona el cicle base.
2. Marca els ṃduls que participen.
3. Marca els RA i activitats concretes.
4. Defineix el nom i la ruta de sortida.
5. Genera el paquet d'alumnes, professorat o tots dos.

Els paquets es generen dins de `Output/<ConfigName>/` amb la mateixa estructura definida en l'especificació.
